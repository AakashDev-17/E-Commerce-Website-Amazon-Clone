## Steps to start the App

<ul>
	<li>Enter you firebase config in ./src/firebase.js</li>
<li>Terminal 1
	<ol>
		<li>npm install</li>
		<li>npm start</li>
	</ol>
</li>
<li>Terminal 2
	<ol>
		<li>cd functions</li>
		<li>npm install</li>
		<li>firebase emulators:start</li>
	</ol>
</li>
<li>Now use the app in http://localhost:3000/</li>
</ul>
